## css layout

```
css/
│── portfolio/          # Styles for the root page, Portfolio.
    │── styles.css          # For base styles and imports
    │── media.css           # For compatability
    │── animations.css      # For animations
    │── specific-components/    # Specific components
        │── navigation.css      # For Top navigation bar
        │── page-header.css     # For the header of the entire page
        │── section.css         # For the general sections on the page

│── <Some other page>/  # Styles for other pages
    │── styles.css
    │── media.css
    │── animations.css
    │── specific-components/
        │── <something>.css

│── legacy/             # Styles for previous legacy pages
    │── <Some other page>/
        │── <something>.css
    │── <Some other page>/
        │── <something>.css
```
